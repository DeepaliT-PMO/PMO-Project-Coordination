# CRM Delivery & Reporting Dashboard (PMO Case Study)

## Overview
This project simulates PMO delivery support for a SaaS-based CRM platform, 
focused on delivery reporting, resource allocation, workforce utilization, 
and risk visibility.

The objective was to create management-ready reporting artifacts and dashboards 
to support delivery governance and decision-making.

## Key Focus Areas
- PMO delivery reporting
- Resource allocation & utilization (WFM)
- Planned vs Actual effort tracking
- Risk & blocker visibility
- Management dashboarding

## Artifacts Included
- Resource Allocation Matrix (Excel)
- Weekly Delivery Status Report (Excel)
- Risk & Blockers Register
- Power BI Delivery Dashboard (PBIX)
- Dashboard snapshot (image)

## Tools Used
- Excel (delivery tracking & allocation)
- Power BI (dashboard & KPIs)
- PMO reporting concepts

## Key Metrics Tracked
- Total tasks vs completed tasks
- Completion percentage
- Effort variance (Planned vs Actual)
- Task status distribution
- High-impact open risks
